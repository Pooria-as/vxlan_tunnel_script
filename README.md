# 🚀 VXLAN Tunnel Management Suite

A powerful, all-in-one VXLAN tunnel management script with real-time monitoring, traffic statistics, and iptables integration.

![Version](https://img.shields.io/badge/version-3.0-blue)
![Bash](https://img.shields.io/badge/bash-5.0+-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## ✨ Features

- 📊 **Real-time Monitoring** - Live traffic statistics with transfer rates
- 📈 **Human-readable Traffic** - Auto-formats KB/MB/GB
- 🚀 **One-Command Setup** - Quick install and run
- 🔒 **iptables Integration** - Auto-configures firewall rules
- 🌐 **Connection Testing** - Test tunnel with ping and wget
- 🎨 **Beautiful UI** - Color-coded, user-friendly interface
- ⚡ **Dynamic Detection** - Auto-detects network interfaces and IPs
- 📝 **Configuration Persistence** - Saves settings for reuse
